# 🧮 Calculadora de IMC

Uma calculadora simples e moderna de **IMC (Índice de Massa Corporal)** desenvolvida com HTML, CSS e JavaScript. O projeto permite ao usuário inserir seu peso e altura para obter automaticamente o cálculo do IMC e sua classificação.

---

## 🚀 Funcionalidades

* 📏 Entrada de **peso (kg)**
* 📐 Entrada de **altura (m)**
* ⚡ Cálculo automático do IMC
* 📊 Exibição do resultado com **duas casas decimais**
* 💬 Mensagem indicando a classificação:

  * Abaixo do peso
  * Peso ideal
  * Sobrepeso
  * Obesidade

---

## 🖥️ Interface

A interface possui um design moderno em modo escuro, com:

* Layout centralizado
* Ilustração visual
* Campos estilizados
* Botão de cálculo interativo

---

## 🧠 Fórmula do IMC

O cálculo do IMC é feito utilizando a fórmula:

```
IMC = peso / (altura * altura)
```

Exemplo:

```
IMC = 65 / (1.82 * 1.82) = 19.62
```
